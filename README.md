# funEvalExpr

The goal of this project is to implement an math expression parser and evaluate said expressions, in Haskell.

## How to build
```sh
$ make re
```

## Examples
```sh
∼/B-FUN-500> ./funEvalExpr "3 + 5.34"
8.34
∼/B-FUN-500> ./funEvalExpr "(0.345+ 5 )*( -2-1) / 3"
-5.34
```
